# Project 3: SQL Data Analysis

## 📌 Objective
Use SQL queries to extract insights from a real-world dataset.

## 🛠️ Tools Used:-
- PostgreSQL 18
- pgAdmin 4

## 📂 Dataset
- **Name:** cleaned_internship_dataset.
- **Database:** internship_task3.
- **Columns:** Date, CustomerID, Product, Quantity, UnitPrice, PaymentMethod, OrderStatus, ItemsInCart, CouponCode, ReferralSource, TotalPrice.

## 📊 Queries Performed:-

| # | Query Type | Description |
|---|-----------|-------------|
| 1 | SELECT | View all data |
| 2 | WHERE | Filter Delivered orders |
| 3 | WHERE | Orders with UnitPrice > 500 |
| 4 | ORDER BY | Sort by TotalPrice |
| 5 | GROUP BY | Product wise order count |
| 6 | COUNT | Total number of orders |
| 7 | SUM | Total revenue |
| 8 | AVG | Average order value |
| 9 | GROUP BY + SUM | Payment method wise revenue |

## ✅ Key Learnings:-
- Importing CSV into PostgreSQL.
- Writing SELECT, WHERE, ORDER BY, GROUP BY queries.
- Performing aggregations using COUNT, SUM, AVG.
- Extracting business insights from raw data.
